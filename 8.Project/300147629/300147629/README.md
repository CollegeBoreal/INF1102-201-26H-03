Vérifier la disponibilité de sites web et mesurer le temps de réponse.

## 2. Structure du projet

```bash
bash scripts/analyse.sh
```

## 4. Dépendances
- Python >= 3.8
- Module Python : requests

## 5. Résultat attendu
- `output/rapport.txt` : statut et temps de réponse des sites
