Ce dossier contient les fichiers nécessaires pour déployer une machine virtuelle avec OpenTofu et Proxmox.



Fichiers du projet

\- provider.tf : configuration du provider Proxmox

\- main.tf : définition de la machine virtuelle

\- variables.tf : déclaration des variables

\- terraform.tfvars : valeurs des variables



Commandes principales

tofu init

tofu plan

tofu apply
